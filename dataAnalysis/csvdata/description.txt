1. 정규화하지 않은 데이터

closed_group.csv : 지역구 및 폐업연월별 폐업한 점포수
(이건 지역마다 다른 그래프를 띄우니 정규화 필요없을 듯)

whole : 기간별 서울특별시 내 총 배달건수 변화량 절대치

2. 정규화한 데이터

opening_normalized.csv : 개업중인 점포수 / 구별 현재 인구수 * 10000

past_closed_normalized.csv : 코로나 이전에 폐업한 점포수 / (구별 19년 2/4분기 ~ 20년 1/4분기 인구수 평균) * 10000

present_closed_normalized.csv : 코로나 이후에 폐업한 점포수 / (구별 20년 1/4분기 ~ 21년 2/4분기 인구수 평균) * 10000
